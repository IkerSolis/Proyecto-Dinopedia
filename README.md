# Proyecto-Dinopedia

**Dinopedia** es una enciclopedia digital independiente enfocada en la recopilación y presentación estructurada de información detallada sobre diversas especies de dinosaurios. Este sistema ha sido diseñado bajo el estándar **IEEE Std 830-1998** para garantizar una especificación de requisitos robusta y profesional.

---

## Propósito del Proyecto
El objetivo principal es crear una herramienta de referencia digital de fácil acceso, completa y actualizada que apoye propósitos educativos y de interés general. El sistema permite a los usuarios buscar, navegar y visualizar perfiles detallados de dinosaurios sin necesidad de registros o cuentas.

## Funcionalidades Clave

### Para Usuarios Generales
* **Búsqueda y Filtrado:** Localización de especies por nombre común o científico con filtros por era, dieta y ubicación.
* **Perfiles Detallados:** Visualización de nombres, descripciones, periodos geológicos, hábitats y tamaños.
* **Galería Multimedia:** Acceso a imágenes representativas y fósiles con opción de ampliación.

### Para Administradores
* **Panel de Gestión:** Módulo autenticado para agregar, editar o eliminar registros de la base de datos.
* **Control de Carga:** Validación de formatos (JPG/PNG) y tamaño de archivos multimedia.

---

## Stack Tecnológico
* **Frontend:** HTML5, CSS3 y JavaScript con diseño responsivo.
* **Backend:** Capa intermedia segura para comunicación con la base de datos.
* **Base de Datos:** SQL estructurado.
* **Seguridad:** Protocolo de transferencia segura HTTPS.
